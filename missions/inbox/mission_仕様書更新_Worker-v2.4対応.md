### Step 2: Worker仕様書の更新
以下の内容を仕様書に反映:

#### 新コマンド（v2.1追加）
| コマンド | 別名 | 動作 | 返信形式 |
|---------|------|------|---------|
| ヘルプ | help, ?, ？ | コマンド一覧表示 | テキスト |
| ヘルプ 指示 | ヘルプ指示 | 指示の送り方ガイド | テキスト |
| 読む ○○ | read ○○ | ファイル内容LINE表示（5000字制限） | テキスト |
| 状態 | ポストマン, postman, ステータス | システム情報（v2.1でエイリアス追加） | テキスト |

#### Flex Message対応（v2.2追加）
- カプセルコマンド → Flex Messageボタン式で返信
- フォルダ ○○ コマンド → Flex Messageボタン式で返信
- ファイル名タップで「読む パス/ファイル名」が自動送信

#### ページネーション（v2.3追加）
- カプセルコマンドが「カプセル N」形式でページ指定可能
- 1ページ10件表示
- 「もっと見る」ボタン（緑色、style: primary）で次ページ

#### 日付抽出ソート（v2.4追加）
- `extractDateFromName()` — ファイル名のどこにあってもYYYY-MM-DDを正規表現抽出
- `shortenFileNameOnly()` — 日付・拡張子を除去してファイル名のみ返す
- `shortenDisplayName()` — 📝 MM-DD ファイル名 形式（後方互換用）
- ソート順: 日付あり→日付降順、日付なし→末尾（ファイル名降順）
- daily/master/plansセクション別表示をやめ、全ファイル日付順で混合表示
- Flex Messageの日付別行表示: 緑色の日付見出し（📅 MM-DD）+ ファイル名ボタン
- 同じ日付は1回だけ見出し表示
- ヘッダー内訳: 📅22 📚13 📋2 形式（短縮）

#### LINEコマンド一覧（v2.4時点・完全版）
| コマンド | 別名 | 動作 | 返信形式 |
|---------|------|------|---------|
| 状態 | ポストマン, postman, ステータス | システム情報表示 | テキスト |
| ヘルプ | help, ?, ？ | コマンド一覧 | テキスト |
| ヘルプ 指示 | ヘルプ指示 | 指示の送り方ガイド | テキスト |
| カプセル | capsule | カプセル保管庫（日付順Flex） | Flex Message |
| カプセル N | capsule N | カプセルN番目ページ | Flex Message |
| アイデア一覧 | ideas, アイデア | アイデア保管庫 | テキスト |
| フォルダ一覧 | folders | 全フォルダ構造 | テキスト |
| フォルダ ○○ | — | 指定フォルダ中身（日付別行Flex） | Flex Message |
| 読む ○○ | read ○○ | ファイル内容表示 | テキスト |
