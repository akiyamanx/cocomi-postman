## 🔧 Tech Notes 追記

### Flex Message活用パターン
- 返信にtype: "flex"のbubbleコンテナを使用
- ファイル名をbuttonコンポーネントで表示
- action type: "message"でタップ→テキスト自動送信→コマンド実行
- 「もっと見る」ボタン: style: "primary" + color: "#1a8f5c"
- size: "mega"で横幅最大

### ページネーションパターン
- コマンド名＋数字: 「カプセル 2」で2ページ目
- 正規表現判定: `/^(コマンド名)(\s+(\d+))?$/`
- 1ページ10件、タイトルに「(1〜10 / 全34件)」表示
- 最終ページでは「もっと見る」ボタン非表示

### ファイル名降順ソート
- `files.sort((a, b) => b.name.localeCompare(a.name))`
- 日付入りファイルは自動的に新しい順に
- API追加呼び出しゼロ

### v2.4 日付抽出ソート（命名規則混在対応）
- `extractDateFromName()` でファイル名のどこにあっても YYYY-MM-DD を正規表現抽出
- 日付あり→日付降順、日付なし→末尾に配置
- daily/master/plans混合で全ファイル日付順表示

### v2.4 Flex Message日付別行表示
- 日付グループの緑見出し（📅 MM-DD）＋ファイル名ボタンの2段構成
- `shortenFileNameOnly()` で日付・拡張子除去したファイル名のみをボタンlabelに
- 同じ日付は1回だけ見出し表示（lastDate変数で重複回避）
