# 6. 📝 技術DIFF（コード変更の詳細）

## COCOMIビートパッド（cocomi-beat-pad.jsx）
```
React + Tone.js アーティファクト
├── 音源: 9種類のシンセ（MembraneSynth, NoiseSynth, MetalSynth, Synth）
├── UI: 3x3パッドグリッド（タップで即発音、ネオンエフェクト）
├── シーケンサー: 16ステップ × 9パッド（タップでON/OFF）
├── ループ再生: Tone.Loop + Transport（BPM 80〜160可変）
├── スコアシステム: タップでポイント＋コンボボーナス
├── デザイン: ダークネオン風、COCOMI CORE🤍フッター
└── プラットフォーム: Claude上で直接実行（.jsx アーティファクト）
```

## PWAアイコン追加
- icon-192.png（192x192）、icon-512.png（512x512）を新規作成
- Python Pillowで生成: 青背景(#1a56db) + 白マップピン + 「MAP」テキスト
- manifest.jsonの参照先（icon-192.png / icon-512.png）と一致
- これによりChrome「アプリをインストール」が表示されるようになった

## manifest.json
- 変更なし（元から正しく設定されていた）
- name: "メンテナンスマップ v2.2"、display: "standalone"、アイコン定義済み
