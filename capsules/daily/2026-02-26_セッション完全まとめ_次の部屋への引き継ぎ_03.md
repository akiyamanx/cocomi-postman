## 🔑 今回確定した技術知見

### 1. LINEリッチメニューの設定方法
- Messaging APIで3ステップ: ①リッチメニュー定義作成→②画像アップロード→③デフォルト設定
- 各ボタンのactionにtype: "message"を設定すると、タップでテキスト自動送信
- setup-richmenu.shでワンコマンド登録。config.jsonからトークン自動読み込み
- 画像サイズ: 2500×1686px（大）または 2500×843px（小）

### 2. Flex Messageの活用パターン
- type: "flex" でbubbleコンテナを返信
- buttonコンポーネントのaction type: "message" でタップ→テキスト送信→コマンド実行
- ファイル一覧のボタン化: ファイル名をlabel、「読む パス/ファイル名」をtextに設定
- サブフォルダのボタン化: 「フォルダ パス/サブフォルダ」をtextに設定（掘り下げ可能）
- size: "mega" で横幅最大のバブルを使用

### 3. ページネーションパターン
- コマンド名＋数字でページ指定:「カプセル 2」「カプセル 3」
- 正規表現で判定: `/^(カプセル)(\s+(\d+))?$/`
- 「もっと見る」ボタン: buildFileListFlexにmoreCommandを渡す
- style: "primary" + color: "#1a8f5c" で緑色の目立つボタン

### 4. v2.4 日付抽出ソート（命名規則バラバラ対応）
- `extractDateFromName(name)` でファイル名のどこにあっても `YYYY-MM-DD` を正規表現で抽出
- ソート優先度: 日付あり同士→日付降順 / 日付あり vs なし→ありが先 / 両方なし→ファイル名降順
- daily/master/plansのセクション分けをやめ、全ファイル混合で日付順表示
- API追加呼び出しゼロで実現

### 5. v2.4 Flex Message日付別行表示
- 日付グループの見出し（textコンポーネント、📅 MM-DD、緑色 #1a8f5c）を表示
- その下にファイル名ボタン（日付・拡張子除去済み）を配置
- 同じ日付は1回だけ見出し表示（lastDate変数で管理）
- `shortenFileNameOnly()` で日付と拡張子を除去し、ファイル名だけをボタンlabelに

### 6. GitHubファイル内容のLINE表示
- GitHub Contents APIでBase64エンコードされたファイル内容を取得
- `Uint8Array.from(atob(cleaned), c => c.charCodeAt(0))` + `TextDecoder('utf-8')` でデコード
- LINE返信は5000文字制限があるため、超過時は先頭4800字＋省略表示

## 📋 レシート管理機能 構想メモ（2026-02-26 夜 アキヤ発案）

### やりたいこと
1. **レシート複数枚まとめて撮影** → 1枚の写真に6〜8枚のレシートを並べて撮る
2. **各レシートを個別に切り出し** → Geminiに四隅座標を聞いて射影変換、スキャン風に整形
3. **PDF保存（2パターン）**
   - まとめPDF: スキャナーで取ったように全レシートを1ページに配置（今のパーキング領収書PDFと同じレイアウト）
   - 個別切り出し: 各レシートを1枚ずつ独立した画像として保存
4. **自動データ抽出（Gemini 2.0 Flash）** → 日付・店名・金額・利用目的を自動認識
5. **駐車場レシート専用**: 日付と利用目的のラベルを自動付与（今は手書き）
6. **データベース保存** → 日付・店名・金額・画像のセットで蓄積
7. **検索＆閲覧** → 「2025年11月のレシート見せて」→ リスト表示 → タップでレシート画像表示

### 技術検討
- **画像認識**: Gemini 2.0 Flash（CULOchanで使用中のAPIキーそのまま）
- **レシート切り出し**: Geminiに四隅座標を聞く → Canvas APIで射影変換
- **PDF生成**: jsPDF（CULOchanで実績あり）
- **データ保存候補**:
  - 方法A: IndexedDB（端末内完結、CULOchanと統一、サーバー代ゼロ）
  - 方法B: Supabase等クラウドDB（複数端末対応、500MBまで無料）
  - → 推奨: 方法A（IndexedDB）＋ レシート画像はGitHubバックアップ（COCOMI Postman経由）
- **レシート画像形式**: JPEG（スキャン風、白背景に補正）
- **検索UI**: 月別カレンダー or 日付フィルター → リスト表示 → タップで画像表示

### 対象レシートの種類
- 駐車場領収書（日付＋利用目的の記入が必要）
- ホームセンター等の買い物レシート（材料費経費）
- その他経費レシート
